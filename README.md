# Talks by Cleiton Loiola

This repository contains the source code and materials for my talks.

Each talk is a [Slidev](https://sli.dev/) presentation. Please refer to the `README.md` inside each talk's directory for more specific instructions on how to run it.

## About the Author

These talks are prepared and delivered by **Cleiton Loiola**.

## Development

This project uses [pnpm](https://pnpm.io/) as a package manager. To get started with development:

1.  Install dependencies:
    ```bash
    pnpm install
    ```
2.  Navigate to a specific talk's directory:
    ```bash
    cd 2025-11-05
    ```
3.  Start the presentation:
    ```bash
    pnpm dev
    ```
